# 项目节点记录

## v2.0.0 (2024-04-14)

### 新增功能

#### 1. 管理后台模块 `/admin`
- **数据看板** `/admin/dashboard`
  - 人格总量、本周新增统计
  - 总下载量、总访问量
  - 下载趋势图（近7天）
  - 热门人格TOP 10
  
- **人格管理** `/admin/personalities`
  - 人格列表展示（支持搜索、筛选）
  - 三种审核状态：待审核/已通过/已拒绝
  - 单条操作：预览、审核通过、审核拒绝、删除
  - 批量操作：全选、批量通过
  - 分页功能
  
- **操作日志** `/admin/logs`
  - 记录所有管理操作
  - 支持按操作类型筛选
  - 支持搜索描述内容
  
- **系统设置** `/admin/settings`
  - 安全设置（密码管理、会话超时）
  - 数据库设置（审核模式、数量上限）
  - 通知设置
  - 系统信息展示

- **管理员认证**
  - 登录页面 `/admin/login`
  - 密码保护（环境变量 ADMIN_PASSWORD）
  - 24小时会话有效期
  - localStorage 存储认证状态

#### 2. .skill生态兼容模块
- **SKILL.md解析器** `lib/skillParser.ts`
  - 解析frontmatter格式
  - 自动推断人格属性（角色、性格、标签等）
  
- **双向转换器** `lib/personalityConverter.ts` (扩展)
  - `skillToPersonality()` - SKILL.md转AgntSoul JSON
  - `personalityToSkill()` - AgntSoul JSON转SKILL.md
  - `detectAndParse()` - 自动检测文件类型并解析

- **上传页面更新** `app/upload/page.tsx`
  - 支持上传JSON、MD、SKILL格式
  - 自动识别文件类型
  - SKILL格式转换预览
  - 格式说明引导

### 数据库更新

#### 新增字段
- `personalities.status` - 审核状态（pending/approved/rejected）

#### 新增表
- `admin_logs` - 管理员操作日志表
  - action: 操作类型
  - target_type: 目标类型
  - target_id: 目标ID
  - description: 操作描述
  - old_value/new_value: 变更值
  - created_at: 创建时间

#### RLS策略更新
- 查询条件增加 `status = 'approved'`

### 环境变量更新
```env
ADMIN_PASSWORD=your_admin_password
```

### 技术实现细节

#### 文件结构
```
app/
├── admin/
│   ├── layout.tsx          # 管理员布局（含侧边栏）
│   ├── login/page.tsx     # 登录页
│   ├── dashboard/page.tsx  # 数据看板
│   ├── personalities/page.tsx  # 人格管理
│   ├── logs/page.tsx       # 操作日志
│   └── settings/page.tsx   # 系统设置
├── api/admin/
│   ├── verify/route.ts           # 认证API
│   ├── personalities/route.ts     # GET人格列表
│   ├── personalities/[id]/route.ts # PATCH状态/DELETE删除
│   └── logs/route.ts              # 日志查询
lib/
├── skillParser.ts          # SKILL.md解析器

└── adminAuth.ts            # 管理员认证工具
```

### 部署说明

1. **更新数据库**
   ```bash
   # 在Supabase SQL Editor执行
   database/schema.sql
   ```

2. **配置环境变量**
   ```bash
   # 在Vercel环境变量中添加
   ADMIN_PASSWORD=你的管理密码
   ```

3. **访问管理后台**
   ```
   https://你的域名/admin
   ```
