# AgntSoul 域名绑定教程

> 目标：将 agntsoul.com 绑定到 Vercel 项目
> 预计耗时：10-30分钟（DNS生效可能需要几小时）

---

## 第一步：在 Vercel 添加域名

### 1.1 打开域名设置页面
1. 登录 Vercel：https://vercel.com
2. 进入项目：`agntsoul`
3. 点击顶部菜单 **Settings**
4. 左侧选择 **Domains**

### 1.2 添加域名
1. 在输入框中输入：`agntsoul.com`
2. 点击 **Add** 按钮
3. Vercel 会显示需要配置的 DNS 记录

---

## 第二步：配置 DNS 解析（火山引擎）

### 2.1 打开火山引擎域名控制台
1. 登录火山引擎：https://www.volcengine.com
2. 进入 **域名服务** 控制台
3. 找到 `agntsoul.com`，点击进入

### 2.2 进入解析设置
1. 点击 **解析设置** 或 **DNS解析**
2. 你会看到现有的解析记录列表

### 2.3 添加解析记录

**Vercel 通常需要添加以下记录**（具体看 Vercel 显示的要求）：

#### 记录一：主域名（@）
| 项目 | 值 |
|------|-----|
| **类型** | `A` |
| **主机记录** | `@` |
| **记录值** | `76.76.21.21` |
| **TTL** | 默认（或600） |

#### 记录二：www子域名
| 项目 | 值 |
|------|-----|
| **类型** | `CNAME` |
| **主机记录** | `www` |
| **记录值** | `cname.vercel-dns.com` |
| **TTL** | 默认（或600） |

### 2.4 操作步骤
1. 点击 **添加记录**
2. 选择记录类型（A 或 CNAME）
3. 填写主机记录和记录值
4. 点击 **确认**

---

## 第三步：验证配置

### 3.1 回到 Vercel Domains 页面
添加完 DNS 记录后：
1. 回到 Vercel → Settings → Domains
2. 查看域名状态

### 3.2 状态说明

| 状态 | 含义 | 操作 |
|------|------|------|
| **Pending** | DNS 正在生效 | 等待几分钟到几小时 |
| **Valid Configuration** | 配置成功 ✅ | 无需操作 |
| **Invalid Configuration** | 配置错误 | 检查 DNS 记录是否正确 |

### 3.3 强制检测
如果状态一直是 Pending：
1. 点击域名右侧的 **刷新图标**
2. 或等待 DNS 自动生效（最长48小时，通常几分钟）

---

## 第四步：SSL 证书配置

### 4.1 自动配置
Vercel 会自动为你的域名配置 SSL 证书：
- 当域名状态变为 **Valid Configuration** 后
- Vercel 自动申请 Let's Encrypt 证书
- 通常几分钟内完成

### 4.2 验证 SSL
1. 等待状态变为 🔒 锁图标
2. 访问 `https://agntsoul.com`
3. 浏览器地址栏显示锁图标 = SSL 成功

## 常见问题

### Q1: DNS 记录值是多少？
Vercel 的 IP 地址可能变化，**以 Vercel Domains 页面显示的为准**。常见值：
- A 记录：`76.76.21.21` 或 `76.76.21.9`
- CNAME：`cname.vercel-dns.com`

### Q2: 提示"记录冲突"怎么办？
如果火山引擎提示冲突：
1. 删除已有的相同主机记录
2. 再添加新记录

### Q3: 要不要改 DNS 服务器？
**不需要**。你只需要添加解析记录，不用改 DNS 服务器。

如果 Vercel 提示改 Nameserver（如 `ns1.vercel-dns.com`），那是另一种方式：
- 更快生效
- 但需要在火山引擎改 DNS 服务器
- 改了之后火山引擎的解析设置就失效了

### Q4: 多久生效？
- 国内 DNS 通常 **几分钟到几小时**
- 最长 **48小时**（极少数情况）
- 可以用 `ping agntsoul.com` 测试是否生效
