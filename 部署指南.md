## 第一步：注册 Supabase（数据库）

### 1.1 注册账号
1. 打开 https://supabase.com
2. 点击 "Start your project"
3. 用 GitHub 账号登录（推荐）或邮箱注册

### 1.2 创建项目
1. 点击 "New Project"
2. 填写信息：
   - **Name**: `agntsoul`（或任意名称）
   - **Database Password**: 自己设一个，记下来！（比如：`AgntSoul2026!`）
   - **Region**: 选择 `Northeast Asia (Tokyo)`（离中国最近）
3. 点击 "Create new project"
4. 等待约2分钟，项目初始化完成

### 1.3 获取数据库连接信息
1. 项目创建完成后，点击左侧 "Settings" → "Database"
2. 找到 "Connection string" 部分
3. 复制以下信息（后面要用）：
   - **Project URL**: 在 Settings → API → Project URL
   - **Anon Key**: 在 Settings → API → Project API keys → anon public
   - **Service Role Key**: 在 Settings → API → Project API keys → service_role（⚠️保密！）

### 1.4 初始化数据库表
1. 点击左侧 "SQL Editor"
2. 点击 "New query"
3. 复制 `AI人格平台MVP/database/schema.sql` 的全部内容
4. 粘贴到编辑器中
5. 点击右下角 "Run" 执行
6. 看到 "Success. No rows returned" 表示成功

### 1.5 验证表创建成功
1. 点击左侧 "Table Editor"
2. 应该能看到 `personalities` 表
3. 表里有这些字段：id, name, type, creator, description, personality_json, tags, downloads, created_at

## 第三步：部署到 Vercel

### 3.1 注册 Vercel
1. 打开 https://vercel.com
2. 点击 "Sign Up"
3. 选择 "Continue with GitHub"（用GitHub登录最方便）

### 3.2 导入项目
1. 登录后点击 "Add New..." → "Project"
2. 在 "Import Git Repository" 找到你的 `agntsoul` 仓库
3. 点击 "Import"

### 3.3 配置环境变量
在部署前，需要添加环境变量：

1. 展开 "Environment Variables" 部分
2. 添加以下变量：

| Name | Value | 说明 |
|------|-------|------|
| `NEXT_PUBLIC_SUPABASE_URL` | 你的Project URL | 从Supabase获取 |
| `NEXT_PUBLIC_SUPABASE_ANON_KEY` | 你的Anon Key | 从Supabase获取 |
| `SUPABASE_SERVICE_ROLE_KEY` | 你的Service Role Key | 从Supabase获取，⚠️保密 |

3. 点击 "Add" 确认每个变量

### 3.4 部署
1. 其他设置保持默认
2. 点击 "Deploy"
3. 等待约2-3分钟
4. 看到 🎉 庆祝动画表示部署成功！

### 3.5 获取临时域名
1. 部署成功后，Vercel会给你一个临时域名，类似：
   - `agntsoul.vercel.app` 或
   - `agntsoul-你的用户名.vercel.app`
2. 点击这个链接，验证网站能正常访问

## 第四步：绑定自定义域名

### 4.1 在 Vercel 添加域名
1. 在项目页面点击 "Settings"
2. 左侧选择 "Domains"
3. 输入 `agntsoul.com`
4. 点击 "Add"

### 4.2 配置 DNS 解析
Vercel会显示需要配置的DNS记录，有两种方式：

**方式一：使用火山引擎DNS（推荐）**
1. 回到火山引擎域名控制台
2. 找到 agntsoul.com → "解析设置"
3. 添加 Vercel 提供的 A 记录或 CNAME 记录
4. 通常需要添加：
   - 类型：`A`
   - 主机记录：`@`
   - 记录值：`76.76.21.21`（Vercel的IP，具体看Vercel提示）

**方式二：把DNS改成Vercel的（更快但需要改NS）**
1. 在火山引擎修改DNS服务器为：
   - `ns1.vercel-dns.com`
   - `ns2.vercel-dns.com`
2. 等待DNS生效（最多48小时，通常几分钟到几小时）

### 4.3 等待生效
1. 回到 Vercel Domains 页面
2. 等待域名状态从 "Pending" 变成 "Valid Configuration"
3. 看到 🔒 锁图标表示SSL证书已自动配置

### 4.4 访问你的网站
打开浏览器，访问 `https://agntsoul.com` 🎉

## 常见问题

### Q: Vercel 部署失败怎么办？
A: 查看构建日志，通常是依赖问题。确保 `package.json` 在项目根目录。

### Q: 域名解析不生效？
A: DNS生效需要时间，最长48小时。可以先访问 `.vercel.app` 临时域名。

### Q: 数据库连接报错？
A: 检查环境变量是否正确配置，特别注意 `NEXT_PUBLIC_` 前缀。

### Q: 如何查看访问日志？
A: Vercel 项目页面 → Analytics → 可以看到访问数据
